PORTAL DEL COLABORADOR MPS - README ADMINISTRATIVO

Versión: WEB v5

Contenido integrado:
- Calidad / SGC: Política de Calidad, Misión, Visión, Valores, Guía básica para el uso correcto del SGC y Organigrama.
- Personal / RH: Reglamento Interno y notas informativas de gafete, comedor y áreas comunes.
- Seguridad: notas informativas y EPP obligatorio con iconos.
- Comunicados: administrables mediante data/comunicados.csv y data/comunicados.json.
- Buzón del Colaborador: vinculado a Google Forms.

Responsabilidades:
- Administrador del portal: Coordinador del SGC.
- Comunicados: RH / SGC / Dirección.
- Buzón del Colaborador: RH / SGC / Dirección.
- Revisión de buzón y comunicados: quincenal.

Notas de control:
- Mantener respaldo periódico del portal y de las respuestas del buzón.
- No publicar dibujos de cliente, contratos, precios, parámetros críticos, registros con datos personales o información confidencial.
- Cuando se actualice un documento, reemplazar el PDF correspondiente y actualizar data/documentos.json.
